The backrooms

Vous explorez le hangar toutes les étagères ont l'air particulièrement remplie.
Vous vous arrêtez pour voir s'il n'y aurait pas un endroit ou se cacher.
Vous commencez à escalader une étagère quand tout a coup "CRACK!!!" l'étagère se brise.
Votre jambe est transpercée vous perdez beaucoup de sang. Vous saisissez le requin pour qu'il vous soigne.
Votre camarade vous soigne. Mais vous avez un mauvais pressentiment comme s'il ne pouvait plus le faire pour le reste de la journée.

Vous allez vers le hangar avec les etagères moins remplis
